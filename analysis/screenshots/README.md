# AI Data Analyst Screenshots

This directory holds the canonical Bruin AI Data Analyst screenshots that
feed the `/analysis` page of the site.

## Cycle 1 (current)

The eight `Q*.png` files in this directory are **placeholder images**
generated by `scripts/generate_placeholder_screenshots.py` (invoked once by
hand during Cycle 1 setup). They exist only so the `/analysis` page renders
in fixture mode and CI stays green.

Every placeholder carries the text "Placeholder — real screenshot captured
in Cycle 3 from the live AI Data Analyst." so nobody mistakes them for real
analyst output.

## Cycle 3 (launch)

When the live warehouse lands, the canonical prompts listed in
`pipeline/fixtures/analysis_gallery.csv` will be run through the Bruin AI
Data Analyst and the actual screenshots captured — unedited — into this
directory, replacing the placeholders one-for-one by file name.

The matching `web/public/screenshots/*.png` copies are served by the Astro
site; keep the two directories in sync on every screenshot refresh.
